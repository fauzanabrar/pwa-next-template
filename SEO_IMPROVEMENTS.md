# SEO Template Notes

This template includes optional SEO helpers. Update the values before launching a real project.

## Where to edit

- Branding and site URL: `src/config/app.ts`
- SEO metadata settings: `src/config/seo.ts`
- Robots and sitemap routes: `src/app/robots.ts`, `src/app/sitemap.ts`

## Quick checklist

1) Set `NEXT_PUBLIC_SITE_URL` in `.env.example` (and your hosting env) to your real domain.
2) Update `appConfig.name`, `appConfig.description`, and `appConfig.keywords`.
3) Set `seoConfig.verification.google` if you use Google Search Console.
4) Replace `/public/icons` and update `public/manifest.json`.
5) If you do not want SEO output, disable it in `src/config/features.ts`.

## Notes

- `sitemap.xml` and `robots.txt` are generated by Next.js routes when SEO is enabled.
- `metadataBase` is derived from `appConfig.url`.
- OpenGraph/Twitter images use `appConfig.socialImage`.
